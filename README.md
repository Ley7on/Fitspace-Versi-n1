# Fitspace-Versi-n1
Fitspace es una plataforma web de gestión para gimnasios, diseñada con un enfoque modular y escalable. Desarrollada en Django y con BD MySQL, permite administrar clientes, entrenadores, rutinas y más. La aplicación está orientada a mejorar la experiencia del usuario a través de un sistema de seguimiento de progreso y control de asistencia.
